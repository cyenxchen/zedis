# Zedis (Fork)

一个使用 **Rust** 🦀 和 **GPUI** ⚡️ 构建的高性能、GPU 加速的 Redis 客户端

> 本项目 fork 自 [vicanso/zedis](https://github.com/vicanso/zedis)。
>
> 非常喜欢这个客户端的界面设计——基于 GPUI（Zed Editor 背后的渲染引擎）构建，速度极快、界面也很漂亮。但原项目的功能迭代速度对我来说有点慢，提 PR 还需要原作者审核，感觉比较麻烦，所以就自己 fork 了一份来维护，按需添加自己需要的功能。

## 🔀 Fork 新增功能（基于 v0.3.0）

以下是本 fork 相对于原仓库新增的功能：

### Key 管理
- **Key 重命名**：支持内联重命名 Key，使用 Redis `RENAME` 命令
- **多选批量删除**：支持多选 Key 并批量删除
- **Key 导出/导入**：通过 `DUMP`/`RESTORE` 命令导出和导入 Key，兼容 ARDM 的 CSV 格式
- **右键菜单**：在 Key 树中右键可直接复制和删除 Key
- **刷新 Key 列表**：刷新时保留当前的关键字过滤条件

### 编辑器增强
- **高级编辑对话框**：支持格式转换（JSON、MessagePack、Text、Binary）和压缩格式（LZ4、SNAPPY、GZIP、ZSTD）
- **文本可选中**：Key 名称和对话框标题支持选中和复制
- **搜索快捷键**：编辑器内支持 `Cmd+F` / `Ctrl+F` 搜索
- **Protobuf 支持**：自动检测 Protobuf 格式并支持基于 Schema 的解码

### 连接与侧边栏
- **复制连接**：快速复制现有的服务器连接配置
- **连接列表布局切换**：切换已保存连接的列表展示方式
- **编辑连接**：在侧边栏右键菜单中直接编辑连接配置
- **侧边栏提示**：鼠标悬停显示完整的服务器名称
- **右键关闭**：通过右键菜单关闭服务器连接
- **关键字保持**：切换服务器时保留搜索关键字

### 搜索与导航
- **首页搜索框**：首页支持全局搜索，快捷键 `Cmd+F` / `Ctrl+F`
- **添加服务器按钮**：移至搜索栏区域，附带 Tooltip 提示

### 认证
- **预设凭据**：支持预设连接凭据实现自动认证

### 开发体验
- **文件日志**：支持文件日志输出，方便调试

## 📖 简介

**Zedis** 是为追求速度的开发者设计的下一代 Redis GUI 客户端。

与处理大数据集时容易感到卡顿的基于 Electron 的客户端不同，Zedis 基于 **GPUI**（驱动 [Zed Editor](https://zed.dev) 的同一渲染引擎）构建。这确保了原生的、60 FPS 的流畅体验，即使在浏览数百万个键时，内存占用也极低。

## 📦 安装方式

从 [GitHub Releases](https://github.com/cyenxchen/zedis/releases) 页面下载最新的预构建安装包。

支持平台：**macOS**（aarch64 / x86_64）、**Windows**、**Linux**。

## ✨ 功能特性

### 🚀 极速性能
- **GPU 渲染**：所有 UI 元素均在 GPU 上渲染，带来丝般顺滑的性能体验。
- **虚拟列表**：利用虚拟滚动和 `SCAN` 迭代，高效处理包含 10 万+ 键的列表。

### 🧠 智能数据查看器
Zedis 自动检测内容类型 (`ViewerMode::Auto`) 并以最适合的格式进行渲染：
- **自动解压**：透明地检测并解压 **LZ4**, **SNAPPY**, **GZIP** 和 **ZSTD** 数据，让您可以直接查看实际内容（例如，压缩的 JSON 会被自动解包并格式化显示）。
- **JSON**：自动 **美化（Pretty-printing）** 并支持完整的 **语法高亮**，以获得更好的可读性。
- **MessagePack**：将二进制 MsgPack 数据反序列化为易读的类 JSON 格式。
- **图片**：原生预览存储的图片（`PNG`, `JPG`, `WEBP`, `SVG`, `GIF`）。
- **十六进制视图**：自适应的 8/16 字节十六进制转储 (Hex dump)，用于分析原始二进制数据。
- **文本**：支持大文本显示的 UTF-8 校验视图。

### 🛡️ 安全访问
- **SSH 隧道**：通过堡垒主机安全地访问私有 Redis 实例。支持通过密码、私钥和 SSH 代理进行身份验证。
- **TLS/SSL**：全面支持 SSL/TLS 加密连接，包括自定义 CA、客户端证书和私钥选项。

### 🎨 现代化体验
- **跨平台**：由 GPUI 驱动，Zedis 在 **macOS**、**Windows** 和 **Linux** 上提供一致的高性能原生体验。
- **智能拓扑检测**：自动识别并适配 **单机 (Standalone)**、**集群 (Cluster)** 或 **哨兵 (Sentinel)** 模式。只需连接到一个入口节点，Zedis 即可处理拓扑映射，无需复杂的配置。
- **主题**：预置 **亮色**、**暗色** 和 **跟随系统** 主题。
- **国际化 (I18n)**：全面支持 **英语** 和 **简体中文**。
- **响应式布局**：适应任何窗口尺寸的分栏布局。
